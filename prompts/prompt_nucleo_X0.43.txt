PASO 2 — EDGAR XBRL API (fuente primaria, no leer el 10-K completo)
2a) Obtener CIK:
  GET https://www.sec.gov/files/company_tickers.json
  Header: User-Agent: patonet@example.com
  Buscar el ticker en el JSON → extraer cik_str → formatear a 10 dígitos con ceros: CIK0000000000

2b) Query de datos financieros:
  GET https://data.sec.gov/api/xbrl/companyfacts/CIK[10dígitos].json
  Header: User-Agent: patonet@example.com

2c) Extraer estos conceptos exactos del JSON (us-gaap section):
  REVENUE: probar en orden → Revenues · RevenueFromContractWithCustomerExcludingAssessedTax · SalesRevenueNet
  NET INCOME: NetIncomeLoss · NetIncomeLossAvailableToCommonStockholdersBasic
  CFO: NetCashProvidedByUsedInOperatingActivities
  CAPEX: PaymentsToAcquirePropertyPlantAndEquipment (este tag exacto = Capex real)
  DEUDA LT: LongTermDebt · LongTermDebtNoncurrent
  DEUDA ST: LongTermDebtCurrent · ShortTermBorrowings
  SBC: ShareBasedCompensation
  DIVIDENDOS: PaymentsOfDividendsCommonStock · PaymentsOfDividends
  TOTAL ASSETS: Assets
  GOODWILL: Goodwill
  SHARES: CommonStockSharesOutstanding (DEF14A para insider ownership)

  Filtrar SOLO filings form=10-K, fp=FY. Tomar el más reciente por fecha end.
  Calcular FCF = CFO − Capex.

2d) Para datos no disponibles en XBRL (márgenes, EBITDA, ROIC, PEG):
  Buscar en https://stockanalysis.com/stocks/[TICKER]/ y /financials/

Output: "✓ EDGAR XBRL — FY[AÑO] · [N] conceptos verificados · FCF=$[X]B"
