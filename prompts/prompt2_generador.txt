A. Activacion, estilo y paleta

-   Generar la infografia UNICAMENTE cuando el usuario haya escrito la
    palabra INFOGRAFIA como mensaje independiente, despues de haber

    para valores numericos, sans-serif moderna para etiquetas.

B. REGLAS CRITICAS DE RESPONSIVIDAD — APLICAR A CADA ZONA SIN EXCEPCION

  ESTAS REGLAS SON OBLIGATORIAS EN ABSOLUTAMENTE TODAS LAS ZONAS. Si
  alguna zona no las aplica, la infografia esta incompleta. Revisa zona
  por zona antes de cerrar el HTML.

REGLA R1 — Raiz fluida: el primer selector CSS del documento debe ser
exactamente:

  *, *::before, *::after { box-sizing: border-box; margin: 0; padding:
  0; } html { font-size: clamp(10px, 1.15vw, 17px); } body { width:
  100%; min-height: 100vh; }

Este bloque hace que 1rem se ajuste fluidamente entre 10px y 17px segun
el viewport y el zoom del navegador. Todo lo demas escala en cascada.

REGLA R2 — Unidades permitidas: SOLO rem, em, %, vw, vh, fr. PROHIBIDO
px en font-size, padding, margin, gap, width, height, border-radius. La
unica excepcion son los bordes de linea finos: border: 1px o 2px solid.

REGLA R3 — Contenedores: todos los wrappers de zona usan:

  width: 100%; padding: 1.5rem; overflow: hidden;

REGLA R4 — Grids fluidos: NUNCA definir columnas de ancho fijo en px.
Usar siempre:

  grid-template-columns: repeat(auto-fit, minmax(Xrem, 1fr)); — o —
  grid-template-columns: 1fr 1fr; — o variantes con fr.

REGLA R5 — Canvas y SVG responsivos: todos los graficos (Chart.js, SVG,
canvas) deben tener:

  width: 100%; height: auto; — Para Chart.js ademas: responsive: true,
  maintainAspectRatio: true — Para SVG: viewBox definido, width='100%',
  height='auto'

REGLA R6 — Gauges SVG responsivos: cada velocimetro SVG individual de la
Zona 5 debe declararse con:

  viewBox='0 0 200 120' width='100%' height='auto' style='display:block;
  width:100%; max-width:12rem;'

El tamanio visual del gauge lo controla el contenedor padre via CSS, no
atributos width/height fijos en el SVG.

REGLA R7 — Verificacion final obligatoria: antes de cerrar el bloque
</style>, busca en el CSS cualquier ocurrencia de px que no sea border.
Si la encuentras, sustituye por rem. Si es un width o height, sustituye
por % o fr.

maintainAspectRatio: false es obligatorio cuando el canvas tiene tamano
